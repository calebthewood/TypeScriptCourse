# TypeScript
### Repo for housing code from "Mastering TypeScript" course.

## Setup
Install Globally

``` $ npm install -g typescript ```

Check Version (4.9.5 as of 2/16/2023)

``` $ tsc -v ```

Run a File

``` $ tsc foobar.ts ```

Run a file in "Watch" mode

``` $ tsc -w foobar.ts ```

Compile all .tsc files in a directory

``` $ tsc ```
